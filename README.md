## 1. Valores de las variables
### ¿Cuáles serán los valores de las variables a y b después de cada línea del siguiente algoritmo?

Inicio 

a=" ", b=" "  

   a <- 3  
   a="3", b=" "
  
   escribir("a = " & a)  
   a="3", b=" "
   
   b<-a+5  
   a="3", b="8"
   
   escribir("a = " & a & " y b = " & b)  
   a="3", b="8"
   
   a<-7 
   a="7", b="8"
   
   escribir("a = " & a & " y b = " & b)  
   a="7", b="8"
   
Fin 
a="7", b="8"
